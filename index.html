1. Domain (Pengertian & Struktur)

Pengertian Domain

Domain adalah nama unik yang digunakan untuk mengidentifikasi dan mengakses sebuah website di internet. Domain berfungsi seperti alamat rumah, tetapi untuk website. Tanpa domain, pengguna harus mengetik alamat IP seperti 192.168.1.1 yang sulit diingat.

Contoh domain: google.com, detik.com, tokopedia.com

Fungsi Domain

  • Sebagai alamat website agar mudah diakses.
  • Memudahkan pengguna mengingat situs.
  • Menjadi identitas atau branding untuk bisnis/organisasi.
  • Menunjukkan kategori website melalui ekstensi (misal: .sch, .gov, .edu).

Struktur Domain

Struktur domain terdiri dari beberapa bagian yang membentuk satu alamat lengkap. Format umumnya: 

www.subdomain.domainutama.ekstensi

1. Subdomain

Bagian yang muncul sebelum nama domain utama. Contoh: www.google.com, blog.tokopedia.com. Subdomain bersifat opsional.

2. Domain Utama (SLD)

Bagian inti dari sebuah domain. Contoh: google, detik, liputan6. Inilah yang biasanya dibeli oleh pemilik situs.

3. Ekstensi Domain (TLD)

Bagian akhir domain yang menunjukkan kategori atau negara.

  • .com → commercial
  • .org → organisasi
  • .net → jaringan
  • .sch → sekolah
  • .gov → pemerintah
  • .edu → pendidikan
  • .id → Indonesia

Contoh Struktur Lengkap

  • www.liputan6.com → subdomain: www, domain utama: liputan6, ekstensi: .com
  • news.google.com → subdomain: news, domain utama: google, ekstensi: .com
  • sekolahku.sch.id → domain utama: sekolahku, ekstensi: .sch.id

2. Hosting (Pengertian, Fungsi, Tujuan)

Hosting adalah layanan penyimpanan online untuk menempatkan file website seperti HTML, CSS, gambar, dan database agar website dapat diakses melalui internet.

Fungsi Hosting

  • Menyimpan file dan data website.
  • Membuat website online 24 jam.
  • Menyediakan bandwidth, email, dan keamanan.

Tujuan Hosting

  • Agar website dapat diakses secara global.
  • Memastikan website stabil dan cepat.

3. Ekstensi Domain

  • .com – Commercial (komersial)
  • .org – Organization
  • .sch – Sekolah
  • .gov – Pemerintah
  • .edu – Pendidikan
  • .id – Indonesia

5. Deploy ke GitHub (Part 1)

  1. Login ke www.github.com
  2. Buat repository sesuai nama domain
  3. Cek apakah nama repository tersedia (warna hijau = tersedia)
  4. Pilih “Create repository”
